## How to start

**Node.js version 18 or higher is required.**

### Yarn Berry 
```
npm install yarn -g
yarn set version berry
yarn
```

### Projects Run(dev)
```
yarn dev
```

### Projects Run(product)
```
yarn build
yarn start
```
